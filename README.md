# Smart Hospital Emergency Room Analytics System

## Overview
This project is a web-based Hospital ER Analytics System developed using Python, Flask, HTML, CSS, JavaScript, Pandas, and OpenPyXL.

## Features
- Patient Analytics
- Doctor Workload Analysis
- Bed Occupancy Analysis
- Waiting Time Prediction
- Excel Report Generation
- Interactive Dashboard

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Pandas
- OpenPyXL

## Installation

Clone the repository

git clone github.com/tanishkasharma0204-svg/Hospital-ER-Analytics-System

Move into project folder

cd Hospital-ER-Analytics-System

Install requirements

pip install -r requirements.txt

Run

python app.py

Open browser

http://127.0.0.1:5000
